- Functional unit
## It is the same for all scenarios
## It is a quantified and additive value, NOT a ratio!
## If a component is studied (e.g. plane engine), FU generally refers to the overall function (the plane)
## You can arbitrarily select the amount of functional unit you fix as a reference for comparison
## When what you need to buy per functional unit changes as the service duration is extended, you need to make the service duration explicit in the functional unit
## Some functional units already implicitly account for the duration of the service (e.g. 1kWh)
## The functional unit is the common unit representing the function of the system (the offered service), providing the basis for scenario comparison. In the inventory, emissions are calculated per functional unit.
