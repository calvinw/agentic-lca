- Summary
- A product system description includes unit processes, intermediary product flows within the system and, elementary flows from and to the environment  as well as the product and its function crossing the system boundaries to the economy.
- The process tree is built starting from the references flows (what was purchased to obtain the product function) and moving through the different tiers of processes.
- The process tree is documented using intermediary flows to calculate the total amount of each process per functional unit.
